# ✈️ Flight Passenger Analytics Dashboard

## Project Overview

This project is an interactive data visualization dashboard built using Streamlit, Pandas, Matplotlib, and Seaborn.

The dashboard analyzes airline passenger data from 1949 to 1960 and provides useful insights through charts, filters, and KPI cards.

---

## Features

### KPI Cards

* Total Records
* Total Passengers
* Average Passengers
* Maximum Passengers

### Interactive Filters

* Year Filter
* Month Filter
* Passenger Range Filter

### Visualizations

1. Line Chart
2. Bar Chart
3. Histogram
4. Pie Chart
5. Scatter Plot
6. Box Plot
7. Heatmap
8. Area Chart
9. Count Plot
10. Violin Plot

### Additional Features

* Dataset Preview
* Download Filtered Data
* Responsive Layout
* Interactive Dashboard

## Installation

Clone the project:

git clone <repository_url>

Move into project folder:

cd flight_passenger_dashboard

Install dependencies:

pip install -r requirements.txt

---

## Run Dashboard

streamlit run app.py

## Dashboard Insights

### Trend Analysis

Passenger traffic consistently increased from 1949 to 1960.

### Peak Season

July and August show the highest passenger volume.

### Growth Pattern

A strong upward trend is visible throughout the years.

### Distribution

Passenger values become increasingly spread out in later years.
